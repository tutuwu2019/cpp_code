# c++ 流(stream) 开发


无论数据来自键盘、文件还是字符串，处理方式都可以统一。

## 流的类层次
```text
ios_base
   └── ios
        ├── istream  (输入流)
        │    ├── ifstream     (文件输入)
        │    └── istringstream (字符串输入)
        ├── ostream  (输出流)
        │    ├── ofstream     (文件输出)
        │    └── ostringstream(字符串输出)
        └── iostream (双向)
             ├── fstream
             └── stringstream

```


## 三大流

### 标准IO流 stream

```cpp
#include <iostream>
using namespace std;

int main() {
    int age;
    cout << "请输入年龄: ";
    cin >> age;
    cout << "年龄是 " << age << endl;
    cerr << "错误信息(不缓冲)" << endl;
}
```


### 文件流 fstream

```cpp
#include <fstream>
#include <string>

// 写文件
ofstream fout("data.txt");
if (fout.is_open()) {
    fout << "Hello" << endl;
    fout.close();
}

// 读文件
ifstream fin("data.txt");
string line;
while (getline(fin, line)) {
    cout << line << endl;
}

// 二进制模式
ofstream bout("data.bin", ios::binary);
int x = 42;
bout.write(reinterpret_cast<char*>(&x), sizeof(x));


```


### 字符串流

```cpp
#include <sstream>

// 字符串 -> 数值
string s = "123 45.6";

int a; double b;
iss >> a >> b;

// 数值 -> 字符串

```

### 常规格式控制

```cpp
#include <iomanip>

cout << hex << 255;              // ff
cout << setw(10) << 42;          // 宽度10,右对齐
cout << setfill('0') << setw(4) << 7;  // 0007
cout << fixed << setprecision(2) << 3.14159;  // 3.14
cout << left << setw(8) << "abc";       // 左对齐
cout << boolalpha << true;       // 输出 true 而非 1

```
### 状态检查

流有4个状态标志，开发时必须检查

```cpp
ifstream f("x.txt");
if (!f)              f.good();  // 一切正常
if (f.eof())  ...    // 到达末尾
if (f.fail()) ...    // 格式错误
if (f.bad())  ...    // 致命错误

// 典型读取循环
int n;
while (cin >> n) {   // 自动检查状态
    // 处理 n
}
cin.clear();         // 清除错误状态
cin.ignore(1000, '\n'); // 丢弃错误输入

```


### 自定义类支持流操作

通过重载 << 和 >>

```cpp
class Point {
    int x, y;
public:
    Point(int a=0, int b=0) : x(a), y(b) {}
    
    friend ostream& operator<<(ostream& os, const Point& p) {
        return os << "(" << p.x << "," << p.y << ")";
    }
    friend istream& operator>>(istream& is, Point& p) {
        return is >> p.x >> p.y;
    }
};

Point p(3, 4);
cout << p;   // 输出 (3,4)


```


### 自定义缓冲区

集成 streambuf 可以实现自己的流(比如日志流、网络流、压缩流)

```cpp
class LogBuf : public std::streambuf {
protected:
    int_type overflow(int_type c) override {
        if (c != EOF) {
            // 自定义处理,比如加前缀、写到多个目的地
            std::cout.put('[');
            std::cout.put(c);
        }
        return c;
    }
};

```

## 流开发经典场景

> 逆向修改动态库，改变一些汇编指令，具体就可以通过流开发，流打开文件、定位、修改、关闭


### 文件读写与日志系统

```cpp
// 简易日志系统
class Logger {
    std::ofstream log_;
public:
    Logger(const std::string& f) : log_(f, std::ios::app) {}
    
    template<typename T>
    Logger& operator<<(const T& msg) {
        auto t = std::time(nullptr);
        log_ << std::put_time(std::localtime(&t), "[%F %T] ") << msg;
        return *this;
    }
};

Logger lg("app.log");
lg << "系统启动\n";

```


### 字符串与数值互转
stringstream 是 C++ 里最经典的类型转换工具,比 atoi/sprintf 更安全。

```cpp
// 把一行 CSV 解析成字段
std::string line = "Alice,30,88.5";
std::stringstream ss(line);
std::string name; int age; double score;
char comma;
ss >> name >> comma >> age >> comma >> score;

// 数值转字符串
std::ostringstream oss;
oss << "ID_" << std::setw(6) << std::setfill('0') << 42;
// 结果: ID_000042

```

### 网络数据报组装与解析

```cpp
// 从接收缓冲区构造流,按协议格式解析
std::string buffer = recv_data();
std::istringstream packet(buffer, std::ios::binary);

uint32_t magic, length;
```

### 大文件分块处理
处理 GB 级文件不能一次读入内存,必须流式分块。

```cpp
std::ifstream fin("huge.bin", std::ios::binary);
constexpr size_t CHUNK = 64 * 1024;  // 64KB
std::vector<char> buf(CHUNK);

while (fin.read(buf.data(), CHUNK) || fin.gcount() > 0) {
    size_t n = fin.gcount();
    process(buf.data(), n);  // 分块处理:哈希、压缩、加密等
}
```


### 内存流/流重定向

```cpp
// 重定向 cout 到字符串,用于测试
std::ostringstream capture;
auto* old = std::cout.rdbuf(capture.rdbuf());

run_function();  // 内部用 cout 输出

std::cout.rdbuf(old);
assert(capture.str() == "expected output\n");

// 重定向到文件
std::ofstream f("out.txt");
std::cout.rdbuf(f.rdbuf());

```
