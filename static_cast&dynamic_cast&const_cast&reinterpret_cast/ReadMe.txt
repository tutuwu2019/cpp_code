一些总结

static_cast：
编译时 类型检查。        重点！！！
用于相关类型之间的转换（基本数据类型、类层次结构中的基类和派生类）。      重点！！！
不进行运行时类型检查。

dynamic_cast：
运行时 类型检查。        重点！！！
用于类层次结构中的安全向下转换。      重点！！！ 基类指向派生类 注意要有虚函数 否则 怎么实现多态？
需要类有至少一个虚函数（开启 RTTI 支持）。

const_cast：
移除或添加 const 属性。    重点！！！
不能用于基本数据类型的常量。

reinterpret_cast：
直接对比特模式进行重新解释。
用于任意指针类型之间的转换。
不安全且依赖于平台。    重点！！！！
