### 掩码
> 在计算机科学中通常指一个用于选择、过滤或者操作特定位的二进制模式。它与位运算紧密结合，是底层变成、系统编程、图像处理等领域的重要工具。


应用场景
- 权限管理，比如linux 文件权限
- 状态标志
- 网络编程，子网掩码
- 图形与图像处理，颜色通道掩码、像素操作
- 数据压缩与编码，位字段、协议解析
- 算法优化，布隆过滤器、位图、状态压缩DP
- 加密与哈希，某些算法(如CRC、MD5)、随机数生成
- 调试与测试，断点掩码、日志级别
