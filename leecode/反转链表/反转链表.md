# 反转链表

> 其实反转链表可以做一个系列的题，因为 有时候反转 一些变形的题
>> - 反转前 N 个节点 / 反转区间
>>   

### leetcode 209 反转链表

> 迭代

采用头插法
> 注意实际开发记得释放内存

```cpp
ListNode* reverseList(ListNode* head) {
        ListNode* pre = nullptr;
        ListNode* curr = head;
        while(curr){
            auto tmp = curr->next;
            curr->next = pre;
            pre = curr;
            curr = tmp;
        }
        return pre;
    }
```
